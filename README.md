# What is Student.AI

An AI-powered study tool built for students, monitored by teachers, enhancing quality of studying all while helping teachers 
through the process enabling to implment their knowledge.

## The Problem

Tools like ChatGPT are too broad for studying. They give you 
answers instead of helping you learn, teachers have no visibility 
into how students use them, and they aren't built around how 
schools actually work.

## The Solution

Student.AI is a web app that gives students an AI tutor that 
guides them toward answers rather than handing them out. Teachers 
get a dashboard to monitor usage, track struggles, and see 
progress in real time. This AI will be design to adapt to student 
learning styles allowing them to succesfully learn and implment 
through this AI

## Planned Features
- AI chat restricted strictly to academic topics
- Subject and grade level customization
- Teacher dashboard with conversation monitoring
- Usage stats per student
- Needed improvement alerts to teachers
- and more...
